284 """ 

285 Optional decorator for case functions so as to customize some information. 

286 

287 ```python 

288 @case(id='hey')

291 ``` 

292 

293 :param id: the custom pytest id that should be used when this case is active. Replaces the deprecated `@case_name`

297 `@case_tags` and `@target` decorators.
